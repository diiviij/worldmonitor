📌 Key Features

  • Location: Header, next to Region Selector - always visible and accessible
  • Provider Dropdown: Quick selection with status indicators (✓ or ○)
  • API Key Input: Direct text field to enter/paste API keys without opening settings
  • Save Button (💾): One-click save or press Enter for quick workflow
  • Toggle Visibility (👁️): Show/hide masked API key for verification
  • Status Indicator: Real-time feedback (green ✓ / yellow ○ / red ✗)
  • Compact Design: Matches header styling, minimal footprint
  • Multi-Provider: Remembers API keys for all providers independently
🎯 Workflow: First-Time Setup
  1. Select provider from dropdown (e.g., "OpenAI")
  2. See status: "○ OpenAI needs API key"
  3. Click "Get Key" link OR paste existing key
  4. Enter API key in input field
  5. Click 💾 Save button (or press Enter)
  6. Status updates: "✓ OpenAI is configured"
  7. Done! Summarization now uses OpenAI first
⚡ Workflow: Switching Providers
  1. Click dropdown, select different provider
  2. Input auto-fills with saved key (masked)
  3. Status shows configuration state instantly
  4. If not configured: enter key and save
  5. If configured: ready to use immediately
  6. System automatically switches primary provider
  7. Fallback chain updated accordingly
