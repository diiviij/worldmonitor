ℹ️ How It Works

  • Your selected provider from the dropdown is tried first
  • If unavailable, automatically falls back through other configured providers
  • Final fallback to browser-based T5 model (works offline)
  • All API calls are cached to minimize costs and improve performance
  • Switch providers anytime without restarting the app
